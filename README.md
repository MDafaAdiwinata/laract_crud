# Laravel CRUD Inertia React  
### with React Starter Kit

## Description
Project ini merupakan aplikasi **CRUD (Create, Read, Update, Delete)** berbasis web yang dibangun menggunakan **Laravel** sebagai backend dan **React + Inertia.js** sebagai frontend.     Project ini bertujuan untuk mempelajari integrasi Laravel dengan React tanpa REST API, serta penerapan **TypeScript** dan **ShadCN UI** dalam pengembangan aplikasi modern.

Aplikasi ini menggunakan konsep **Single Page Application (SPA)** dengan performa yang optimal dan struktur kode yang rapi.
